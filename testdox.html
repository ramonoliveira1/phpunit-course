Avaliador (Service\Avaliador)

  • Avaliador encontra o maior valor de lances with data set "ordem-aleatoria"
  • Avaliador encontra o maior valor de lances with data set "ordem-crescente"
  • Avaliador encontra o maior valor de lances with data set "ordem-decrescente"
  • Avaliador encontra o menor valor de lances with data set "ordem-aleatoria"
  • Avaliador encontra o menor valor de lances with data set "ordem-crescente"
  • Avaliador encontra o menor valor de lances with data set "ordem-decrescente"
  • Avaliador deve buscar os 3 maiores valores with data set "ordem-aleatoria"
  • Avaliador deve buscar os 3 maiores valores with data set "ordem-crescente"
  • Avaliador deve buscar os 3 maiores valores with data set "ordem-decrescente"

Leilao (Model\Leilao)

  • Leilao deve receber lances with data set "dois-lances"
  • Leilao deve receber lances with data set "um-lance"
  • Leilao nao deve receber lances repetidos
  • Leilao nao pode receber mais de 5 lances por usuario
  • Leilao vazio nao pode ser avaliado
  • Leilao finalizado nao pode ser avaliado
